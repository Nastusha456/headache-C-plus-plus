# Запуск linux
1. Скачать библиотеку SFML https://www.sfml-dev.org/download.php
2. Указать путь до скаченной библиотеки в makefile
   ```
   SFMLL = <path to lib>/lib
   SFMLI = <path to lib>/include 
   ```
4. Запустить в терминале
  ```bash
  make run
  ```
